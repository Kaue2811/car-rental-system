# 🚗 Car Rental System

Sistema backend para gerenciamento de aluguel de veículos, desenvolvido em Java com foco em programação orientada a objetos.

## 🚀 Funcionalidades
* Registro de aluguel de veículos
* Cálculo de pagamento por hora ou por dia
* Geração de fatura
* Cálculo de impostos baseado em regras de negócio

## 🛠️ Tecnologias utilizadas
* Java
* API de Data e Hora (`java.time`)

## 🧠 Conceitos aplicados
* Programação Orientada a Objetos (POO)
* Separação de responsabilidades (entities, services, application)
* Uso de interfaces para desacoplamento
* Implementação do padrão Strategy (`TaxService`)

## 💡 Objetivo do projeto
Praticar conceitos de backend e boas práticas de desenvolvimento, simulando regras reais de negócio.

## 💻 Como rodar o projeto localmente

**Pré-requisitos**
* Java 17 (ou superior) instalado na máquina
* Git instalado

**Passo a passo**
1. Clone este repositório em sua máquina local:
   ```bash
   git clone https://github.com/Kaue2811/car-rental-system.git
   ```

2. Abra a pasta do projeto na sua IDE favorita (IntelliJ, Eclipse, VS Code, etc.).

3. Localize e execute a classe principal (`Program.java`).

4. Siga as instruções que aparecerão no console para interagir com o sistema.
